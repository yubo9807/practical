# 转盘菜单

## Option

| 属性名 | 说明 | 类型 | 默认值 | 备注 |
| --- | --- | --- | --- | --- |
| el* | 挂载元素 | `HTMLElement` | - | - |
| width | 画布宽度 | `number` | 默认取父元素的宽度 | - |
| height | 画布高度 | `number` | 默认取父元素的高度 | - |
| controls | 圆盘配置 | `Control[]` | - | - |

## Enevt

| 事件名 | 说明 | 参数 |
| --- | --- | --- |
| refresh | 一般在容器大小改变后调用 | - |
